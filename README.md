# Agentic AI Systems Portfolio

This repository contains multiple AI/ML projects focused on Agentic AI systems, Retrieval-Augmented Generation (RAG), and LLM-based applications.

## Projects Included

### 1. Agentic RAG Assistant
Location:
`agentic-rag-assistant/`

A full-stack RAG system that:
- Processes and chunks documents
- Generates embeddings
- Stores vectors in a vector database
- Retrieves relevant context
- Uses LLMs to generate intelligent responses
- Implements agentic workflow using graph-based architecture
